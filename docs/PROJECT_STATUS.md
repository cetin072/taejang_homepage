# 프로젝트 현황

## 목적

태장 공식 홈페이지의 현재 구조와 공동 개발 기준을 기록합니다.

## 현재 상태

- 사이트 유형: 순수 HTML/CSS/JavaScript 정적 사이트
- 기본 브랜치: `main`
- 운영 배포: Netlify 정적 배포
- 별도 패키지 관리자나 빌드 명령: 없음
- 콘텐츠 목록: `assets/js/content.js` 기반의 일터·활동 데이터

## 현재 페이지와 자산

- 메인: `index.html`
- 일터: `workplace.html`
- 활동: `activities.html`
- 공통 스타일: `assets/css/styles.css`
- 공통 동작: `assets/js/site.js`
- 이미지: `images/`

## 기준 문서 정리 작업의 파일 범위

기존 정적 홈페이지 파일은 현재 기준본으로 유지하며, 이번 프로젝트 기준 문서 정리 작업에서는 기존 HTML, CSS, JavaScript, 이미지 파일 및 `netlify.toml`을 변경하지 않습니다.

이번 작업의 변경 대상은 `AGENTS.md`, `README.md`, `.gitignore` 및 `docs/` 안의 공동 개발 기준 문서입니다.

## 공동 개발 기준

- `main` 직접 수정 금지
- 모든 변경은 기능 브랜치에서 수행
- 변경 사항은 PR과 검토를 거쳐 `main`에 반영
- Netlify 운영 배포·도메인·환경 설정 변경은 사용자 승인 후 진행
- 정적 홈페이지의 기존 파일은 업무 범위를 명확히 한 뒤에만 수정

## 정보 관리 기준

공개 저장소에 민감정보, 주주명부, 주민등록번호, 비공개 계약 자료, 초상권 사용 여부가 확인되지 않은 사진을 저장하지 않습니다. 태장 공식 정보와 대외 표현은 별도 기준 문서에서 관리하고, 게시 전 해당 기준과 대조합니다.

3. 관리자 모드 요구사항과 권한 설계 검토
