# 태장 B2B 홈페이지 콘텐츠 계획

## 1. 홈페이지의 B2B 역할

태장 홈페이지는 투자·출자 모집 페이지가 아니라, 공식 정체성과 실제 사업 범위·협력 가능성을 정확히 보여 주고 적절한 문의로 연결하는 공개 창구입니다.

- 태장의 법인·인증·사업 정체성 소개
- 현재 운영 사업과 준비 사업 구분
- 수행 가능한 업무와 기업 협력 방식 안내
- 실제 일터·활동·자료의 지속적 축적
- 장애인 고용과 ESG·사회공헌의 의미를 과장 없이 설명
- 문의 전환과 상담 준비 정보 제공

## 5. 공개하면 안 되는 정보

- 근로자의 장애 유형·건강정보·개인 연락처
- 고객사 비공개 계약·협상·발주 내용
- 가격·원가·마진·수익 구조·정산 방식
- 승인되지 않은 기업명·로고·인물 사진
- 비공개 생산량·물량·고용 인정 인원
- 주주·지분·계약대금·부담금 절감 수치
- 내부 운영 문제·내부 일정·개인정보
- 검증되지 않은 수치·성과·완료 표현

## 6. 우선순위

### P0 공개 전 필수

- 공식 정보·주소·인증·연락처 재확인
- 사진 안내 8곳을 승인 사진으로 교체하거나 공개 범위 재결정
- 개소 예정·완료 시제와 초대장 링크 전환
- 문의·지도·정책 링크 확인
- 개발 안내·깨진 링크·부적합 사진 참조 제거

### P1 B2B 신뢰 강화

- 공개용 회사소개서 PDF
- 수행 가능한 업무 범위와 상담 준비사항
- 인증·공식 자료 공개본
- 실제 작업·사업장 사진
- 기업 협력 상세 안내와 FAQ 확장

### P2 실적 축적 후

- 승인된 협력·납품 사례
- 기업·기관 후기
- 지역·환경 활동 결과
- 보도자료와 공개 성과

### P3 관리자 시스템 연계

- 채용, 배너, 팝업, 자료실, PDF 교체
- 콘텐츠 승인·예약·게시·보고서
- 문의 배정·처리 상태·회신 기록

## 8. 회사소개서 PDF 연계

- 권장 위치: 메인 기업 협력 영역과 향후 자료실에 동일 최신 파일을 참조합니다.
- 파일명: `taejang-company-profile-public-YYYYMM.pdf`처럼 공개본·발행월을 구분합니다.
- 공개 승인: 대표·내부 검토자가 개인정보·계약·수치·로고 승인을 확인합니다.
- 버전 관리: 관리자에는 현재 공개본 1개를 지정하고 이전 버전은 비공개 보관합니다.
- 이전 파일: 공개 URL을 무기한 유지할 필요가 있는지 결정하고, 필요 시 보관 상태와 대체 파일을 기록합니다.
- 관리자 연계: Phase 2에서 PDF 교체·버전·공개일·승인자를 관리합니다.

## 9. B2B 문의 흐름

현재는 `기업 협력 문의 → 전화 또는 이메일`입니다. 향후 관리자 요구사항에는 다음 개념을 연결하되 이번 단계에서 문의 폼을 만들지 않습니다.

1. 문의 유형
2. 기업명·담당자·연락처·이메일
3. 원하는 물품·업무·활동
4. 예상 일정·예상 물량
5. 참고 파일 첨부
6. 개인정보 처리 동의
7. 내부 담당자 배정
8. 접수·검토·회신·보류·완료 상태
9. 회신 기록과 다음 행동

계약조건·가격·민감한 협상 파일은 공개 콘텐츠 DB와 분리합니다. 문의 관리 기능은 관리자 Phase 4 후보로 두고 보안·보유기간을 먼저 결정합니다.
