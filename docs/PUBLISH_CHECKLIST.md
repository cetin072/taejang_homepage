# 홈페이지 공개 전 최종 검수 체크리스트

대상: 콘텐츠 작성 직원, 홈페이지 반영 담당자, 최종 공개 승인자

이 문서는 자동 검증을 통과한 뒤 사람이 직접 확인해야 하는 공개 전 최종 점검표입니다. 자동 검증만으로 사진 내용·사실관계·승인 여부를 판단할 수 없습니다.

## 1. 글 내용 확인

- [ ] 제목과 요약이 실제 내용과 일치한다.
- [ ] 준비 중·예정·완료 상태가 정확하다.
- [ ] 날짜와 시제가 일치한다.
- [ ] 검증되지 않은 성과·수치가 없다.
- [ ] 실제 계약이나 협력 상태를 과장하지 않았다.
- [ ] 내부 정보, 협상 내용, 계약 조건이 포함되지 않았다.

## 2. 사진 확인

- [ ] [사진 운영 가이드](PHOTO_GUIDE.md)의 지정 파일명을 사용했다.
- [ ] 사진 방향·화질·표시 비율이 적절하다.
- [ ] 얼굴 공개 동의를 확인했다.
- [ ] 기업명·상호·로고 공개 승인을 확인했다.
- [ ] 송장·주소·전화번호·주문정보가 보이지 않는다.
- [ ] 계약서·거래명세표·업무 문서가 보이지 않는다.
- [ ] 근로자의 장애·건강 관련 정보가 노출되지 않는다.
- [ ] 사진과 본문 내용이 일치한다.
- [ ] 같은 사진을 다른 업무 사례처럼 중복 사용하지 않았다.

## 3. 링크와 기능 확인

- [ ] 목록에서 상세 글로 진입한다.
- [ ] 상세 글에서 목록으로 돌아간다.
- [ ] 관련 글 링크가 정상이다.
- [ ] 전화번호·이메일 링크가 정상이다.
- [ ] 네이버지도·카카오맵 링크가 정상이다.
- [ ] 개인정보처리방침·이용약관 링크가 정상이다.
- [ ] 존재하지 않는 `id` 요청 시 안내 화면이 표시된다.
- [ ] 기업 협력·자료실 링크가 정상이며 빈 다운로드 링크가 없다.
- [ ] 자료 상태와 실제 공개 파일의 존재 여부가 일치한다.

## 4. 화면 확인

- [ ] PC 화면을 확인했다.
- [ ] 360px 모바일 화면을 확인했다.
- [ ] 390px 모바일 화면을 확인했다.
- [ ] 제목 줄바꿈이 자연스럽다.
- [ ] 필터 버튼이 잘리지 않는다.
- [ ] 사진 파일명이 안내박스 안에서 정상 줄바꿈된다.
- [ ] 가로 스크롤이 없다.
- [ ] 관련 글 카드가 정상 배치된다.
- [ ] 지도와 푸터 사이 간격이 자연스럽다.
- [ ] 기업 협력 상세·FAQ·자료실 화면을 확인했다.
