# 업무유형별 기록 수준 기준 v1

상태: **확정**

## 1. 목적

태장 업무플랫폼의 기록량을 직급이나 직원의 작성능력으로 일률적으로 정하지 않는다. 업무의 성격, 내용 누락 시 위험, 후속조치 필요성에 따라 기록 수준을 다르게 적용한다.

핵심 원칙은 다음과 같다.

> 현장부서는 업무를 방해하지 않는 최소기록을 하고, 지원·상담·홍보·기획 등 정확성과 누락 방지가 중요한 업무는 충분한 기록을 남긴다. 기록능력이 부족한 직원에게는 기록을 생략하게 하지 않고 구조화된 입력으로 돕는다.

## 2. 1단계: 현장 최소기록

적용 예:

- 생산
- 물류
- 환경정비·공원청소
- 농장
- 기타 현장 작업

기록 목적:

- 업무 누락 방지
- 예외사항 확인
- 다음 반·다음 날 인계
- 재료·장비·안전 문제 확인
- 필요한 지원 연결

기본 항목:

- 결과 상태
- 필요한 경우 수량
- 예외·변경
- 현장 조치
- 후속 확인 여부
- 인계사항
- 선택적 사진

정상 완료 업무에는 긴 설명을 요구하지 않는다. 직원별 분 단위 작업시간, 생산성 순위, 세밀한 행동로그와 자유서술 일일보고는 요구하지 않는다.

## 3. 2단계: 일반 업무기록

적용 예:

- 부서 내 행정업무
- 일반 협업 요청
- 일정 준비
- 자료 취합
- 구매·외부 연락
- 통상적인 사무업무

기본 항목:

- 업무 목적
- 요청 또는 지시 내용
- 담당자
- 기한
- 진행내용
- 결과
- 관련 자료
- 다음 행동

업무의 시작·진행·결과와 책임자가 분명하게 남을 정도로 기록한다.

## 4. 3단계: 전문 상세기록

적용 예:

- 근로자지원
- 상담
- 인사·운영 지원
- 홍보 콘텐츠
- 사업기획
- 영업 및 외부 협의
- 지원사업 조사·신청
- 중요 의사결정 자료

기본 항목:

- 경위와 목적
- 구체적인 사실
- 당사자 또는 관계자 의견
- 판단·검토·협의 내용
- 조치 내용
- 담당자와 기한
- 결과
- 후속 확인
- 변경·검토·승인이력
- 관련 자료

전문 상세기록은 내용 누락을 막되, 민감정보 접근권한을 분리한다.

## 5. 근로자지원·상담 기록 분리

### 제한된 상담·지원 기록

근로자지원 담당자와 책임자 등 허용된 사용자만 확인한다.

- 상담 경위
- 근로자가 표현한 어려움
- 지원 요청과 관련 배경
- 상담자의 관찰과 판단
- 보호자·외부기관 협의
- 민감한 개인정보
- 후속상담 내용

### 현장 전달사항

반장·팀장에게는 업무 수행에 필요한 내용만 전달한다.

- 필요한 작업 조정
- 설명 방법
- 현장에서 확인할 사항
- 조치 기간
- 후속 결과 전달 요청

상담 원문과 진단·장애 세부정보는 현장에 공유하지 않는다.

## 6. 홍보·기획 기록

홍보와 기획 업무는 단순 완료 상태만으로 충분하지 않다.

홍보 기록 예:

- 콘텐츠 목적
- 대상 독자
- 핵심 메시지
- 필수 포함내용
- 참고자료와 사진·영상
- 게시 채널과 예정일
- 개인정보·초상권 주의사항
- 검토·수정 의견
- 승인 내용
- 게시 결과와 후속 활용

기획·외부사업 기록 예:

- 사업명과 주관기관
- 신청대상과 태장 적합성
- 지원내용과 신청기한
- 필수조건과 준비서류
- 검토 의견
- 다음 행동, 담당자와 기한

## 7. 기록 작성 지원

기록능력이 부족한 직원에게 빈 자유서술칸만 제공하지 않는다.

초기 구현 가능한 지원방식:

- 단계별 질문
- 선택항목
- 항목별 쉬운 작성 예시
- 자주 쓰는 문장
- 이전 기록 또는 템플릿 불러오기
- 선택 내용을 문장으로 정리
- 필수항목 누락 경고
- 임시저장
- 팀장 보완 요청
- 저장 전 요약 미리보기

AI 없이 고정 질문과 템플릿만으로 먼저 구현한다.

## 8. 작성과 검토 책임

기록이 중요한 업무는 작성자 혼자 최종 확정하지 않는다.

- 근로자지원·상담: 담당자 작성 → 책임자 검토·보완 → 확정
- 홍보: 직원 작성 → 홍보팀장 검토 → 운영총괄 승인 → 중요 사안만 대표이사 승인
- 일반 사무업무: 담당자 작성 → 필요 시 팀장 확인
- 현장업무: 반장·관리자 최소기록 → 확인 필요 사항만 총괄 공유

검토자는 사실관계, 누락, 담당자와 기한, 개인정보, 다음 행동을 확인한다.

## 9. 기록 수준 결정 기준

기록 수준은 다음 질문으로 결정한다.

1. 내용이 누락되면 근로자 지원, 안전, 품질, 계약, 홍보 또는 대외관계에 문제가 생기는가?
2. 나중에 판단 근거와 협의과정을 확인해야 하는가?
3. 여러 사람이 후속조치를 해야 하는가?
4. 민감정보와 접근권한 분리가 필요한가?
5. 현장작업을 방해하지 않고도 기록할 수 있는가?

직급이 아니라 업무 성격과 위험에 따라 기록 수준을 선택한다.

## 10. 초기 제외

- AI 자동 상담기록 작성
- AI 사실판단과 위험점수
- 근로자 태도·감정·장애정도 평가
- 직원별 기록능력 점수
- 모든 업무의 동일한 긴 보고서
- 모든 기록의 다단계 결재

## 11. 검수 기준

- 현장 최소기록 원칙이 다른 전문업무 기록을 축소하지 않는가
- 전문 상세기록에서 필수 사실과 후속조치가 누락되지 않는가
- 기록이 어려운 직원에게 구조화된 작성 지원을 제공하는가
- 민감한 상담 원문과 현장 전달정보가 분리되는가
- 팀장·책임자의 검토 역할이 형식적인 결재가 아니라 내용 보완을 포함하는가
