# 외부 지도자 제한 접근 및 업무 화면 v1

상태: **확정**

이 문서는 민화작가와 전사 안전교육·외부활동 준비 담당처럼 태장 내부 직원이 아닌 외부·별도계약 지도자의 시스템 접근 범위와 업무 화면을 정의합니다.

## 1. 기본 원칙

- 외부 지도자는 태장 직원·팀장·관리자가 아닙니다.
- 인사정보, 급여, 근로계약, 장애정보, 상담기록, 거래처, 계약, 전체 재고, 경영정보에는 접근할 수 없습니다.
- 본인이 맡은 지도·교육·행사 준비 업무에 필요한 일정, 대상, 준비사항, 기록만 확인합니다.
- 내부 직원용 조직도와 계정 권한에 포함하지 않고 별도의 `외부 지도자` 역할군으로 관리합니다.
- 초기에는 담당 직원이 대신 기록하는 방식을 우선하며, 직접 로그인 필요성이 확인되면 제한 계정을 발급합니다.
- 계약 종료 또는 역할 종료 시 계정을 즉시 정지하고 활성 세션을 종료합니다.

## 2. 공통 첫 화면

외부 지도자가 로그인할 경우 첫 화면에는 다음만 표시합니다.

- 오늘 일정
- 이번 주 일정
- 본인 담당 대상 또는 활동
- 준비해야 할 사항
- 제출 또는 기록이 필요한 내용
- 회사 담당자 연락처

공통 동작은 다음으로 제한합니다.

- 일정 확인
- 준비 완료 표시
- 지도·교육 실시 기록
- 특이사항 전달
- 다음 일정 제안
- 자료 첨부

## 3. 민화 생산 지도 작가

민화작가 화면에는 다음 기능만 제공합니다.

- 방문 지도 일정 확인
- 당일 지도 대상 작업과 작품 확인
- 작품별 진행 단계 기록
- 사용 재료·준비물 요청
- 지도내용과 다음 작업 제안
- 작품 사진 또는 예시자료 첨부
- 제작상 어려움 또는 안전상 주의사항 전달

민화작가는 다음을 할 수 없습니다.

- 근로자 인사평가
- 출퇴근·근무시간 변경
- 생산팀 전체 업무배정
- 직원 상담기록 열람
- 장애·건강정보 열람
- 급여·계약·징계 관련 기록

민화작가가 남긴 내용은 생산 업무관리자 또는 운영총괄이 확인하고 실제 작업배정에 반영합니다.

## 4. 전사 안전교육·외부활동 준비 담당

이 역할은 단순 강사가 아니라 안전교육과 외부활동의 사전 준비·점검·운영 지원을 담당합니다.

화면에는 다음 기능을 제공합니다.

- 주간 안전교육 일정
- 교육 주제와 참여 대상
- 모의훈련 계획
- 외부활동 후보지와 사전답사 일정
- 위험요소·접근성·화장실·휴식공간 점검
- 이동수단·장비·보호구·비상연락망 준비상태
- 행사 당일 안전·참여자 관리 체크리스트
- 교육·훈련·행사 실시 기록
- 사고·근접사고·개선사항 기록
- 다음 교육·활동 준비과제

기본 업무 흐름은 다음과 같습니다.

```text
장소 물색
→ 사전답사
→ 위험·접근성 확인
→ 이동·인원·장비 계획
→ 주 1회 교육·모의훈련
→ 월 1~2회 외부활동
→ 결과·개선사항 기록
→ 다음 활동 준비
```

## 5. 내부 직원과의 연결

- 외부 지도자는 일반 근로자에게 직접 인사·근무 변경 지시를 하지 않습니다.
- 작업 또는 교육상 제안은 지정된 내부 담당자에게 전달합니다.
- 민화 관련 제안은 생산 업무관리자 또는 운영총괄이 검토합니다.
- 안전·외부활동 관련 제안은 운영총괄이 부서별 일정·잔류인원·자원과 함께 조정합니다.
- 직원에게 실제로 배정되는 업무는 내부 팀장 또는 권한을 받은 업무관리자가 등록합니다.

## 6. 접근 제한

외부 지도자는 다음 정보에 접근할 수 없습니다.

- 전체 직원명부
- 주민등록번호·주소·연락처 등 불필요한 개인정보
- 장애 유형과 의료정보
- 상담·고충·지원 기록
- 급여와 근로계약
- 인사변경·휴직·퇴사·징계
- 거래처·계약·단가·매출
- 전체 생산량·재고·주문정보
- 대표이사·운영총괄 전용 보고와 승인

필요한 참여자 명단을 제공할 때도 이름 또는 식별에 필요한 최소 범위만 표시합니다.

## 7. 초기 구현 범위

초기 버전은 외부 지도자의 직접 로그인 없이도 운영할 수 있게 합니다.

- 내부 담당자가 일정 등록
- 지도자가 구두·문자·문서로 전달한 내용을 담당자가 기록
- 지도자료와 결과사진을 담당자가 첨부
- 직접 로그인이 필요한 시점에 한해 제한 계정 활성화

외부 지도자 계정을 먼저 만들어 사용을 강제하지 않습니다.

## 8. 개발 검수 기준

- 외부 지도자에게 내부 직원·팀장 화면이 노출되지 않는가
- 본인 담당 일정과 지도·교육 기록만 확인할 수 있는가
- 민감한 인사·장애·상담·급여정보가 차단되는가
- 외부 지도자의 제안이 내부 책임자의 검토를 거쳐 실제 업무에 반영되는가
- 역할 종료 시 즉시 접근 차단이 가능한가
- 직접 로그인 없이도 내부 담당자 대리 기록으로 초기 운영할 수 있는가
