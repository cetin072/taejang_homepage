# 인사 최소 기능 및 문서 다운로드 v1

상태: 확정
확정 기준: 사용자 기획 대화
적용 범위: 태장 업무 플랫폼의 인사 기능과 공통 다운로드 기능

## 1. 목적

태장 업무 플랫폼의 초기 인사 기능은 복잡한 급여·노무 시스템을 대체하는 것이 아니라, 근로자의 공식 고용 상태와 기본 인사정보를 정확하게 관리하는 데 목적이 있다. 또한 실무상 외부 제출, 회의, 결재, 보관이 필요한 자료는 엑셀 또는 워드 형식으로 내려받을 수 있게 한다.

## 2. 초기 인사 기능 범위

초기 인사 기능은 다음으로 한정한다.

- 입사 등록
- 기본 인사정보 관리
- 소속 부서·직무·근무시간 변경
- 재직 상태 관리
- 휴직·복직·퇴사 처리
- 이력서·근로계약서·입사자료·증빙자료 보관
- 공식 변경이력 보존

급여 계산, 급여명세서, 연차 자동계산, 4대 보험 신고 등은 초기 플랫폼 범위에서 제외하고 기존 방식 또는 외부 서비스를 사용한다.

## 3. 직원명부 엑셀 다운로드

권한을 가진 사용자는 직원명부를 엑셀로 내려받을 수 있다.

기본 항목은 다음과 같다.

- 사원번호
- 이름
- 소속 부서
- 담당 직무
- 입사일
- 재직 상태
- 근무시간
- 연락처
- 비상연락처

현재 화면에서 적용한 검색·필터 조건을 다운로드 결과에도 그대로 반영한다.

예:

- 전체 직원
- 재직자만
- 퇴사자 포함
- 특정 부서
- 특정 근무시간

주민등록번호, 계좌번호, 급여정보, 상담기록 등 민감정보는 직원명부 기본 다운로드에서 제외한다.

## 4. 공통 다운로드 원칙

- 목록·현황·통계·집계 자료는 엑셀로 제공한다.
- 공식 보고·결재·보관 자료는 워드로 제공한다.
- 모든 화면에 일괄적으로 다운로드 기능을 넣지 않는다.
- 실제 사용빈도가 높고 외부 제출·회의·보관 필요가 있는 항목부터 적용한다.
- 화면의 검색·필터 조건을 다운로드 결과에 반영한다.
- 민감정보는 기본 제외한다.
- 다운로드 권한은 역할별로 제한한다.
- 누가, 언제, 어떤 범위의 자료를 내려받았는지 자동 기록한다.

## 5. 초기 우선 엑셀 다운로드

다음 항목을 우선 대상으로 한다.

1. 직원명부
2. 인사변경 이력
3. 근로자지원 활동 현황
4. 후속 확인·미처리 목록
5. 콘텐츠 관리대장
6. 게시 일정표
7. 업무 목록
8. 승인 처리 이력

세부 항목은 각 모듈 기획과 실제 구현 단계에서 조정할 수 있다.

## 6. 초기 우선 워드 다운로드

다음 항목을 우선 대상으로 한다.

1. 근로자 개인 인사요약
2. 근로자지원 특이사항 보고서
3. 근로자별 상담·지원 경과보고서
4. 월간 근로자지원 업무보고
5. 중요 콘텐츠 승인 검토본
6. 경영 현황 요약보고

워드 문서는 화면 내용을 그대로 복사하는 방식이 아니라, 보고·결재·보관에 맞는 정형 문서로 생성한다.

## 7. 기본 다운로드에서 제외할 자료

- 상담 원문 전체 일괄 엑셀
- 주민등록번호·계좌번호·급여정보가 포함된 통합 명부
- 모든 사용자의 열람기록 전체
- 개인별 민감정보 일괄 다운로드
- 첨부파일 전체 일괄 압축 다운로드
- 일반 직원의 전체 회사 인사자료 다운로드

민감자료는 화면 열람보다 다운로드 시 유출 위험이 크므로, 필요 시 별도 권한과 사유 확인 절차를 둔다.

## 8. 개발 원칙

초기에는 다운로드 대상과 양식을 과도하게 늘리지 않는다. 우선순위가 높은 항목부터 구현하고, 실제 운영 중 필요성이 확인되면 추가한다. 새로운 다운로드 기능을 추가할 때는 목적, 이용 권한, 포함 항목, 제외할 민감정보, 다운로드 이력 기록 여부를 함께 검토한다.
