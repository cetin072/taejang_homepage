# 자료·문서·파일 연결 MVP v1

상태: **확정**

## 1. 목적

태장 업무 플랫폼 초기 버전에서는 자체 파일 저장소나 Google API 연동을 개발하지 않는다. 업무·사업·행사 화면에서 필요한 Google Drive 또는 외부 자료 링크를 쉽게 등록하고 열 수 있도록 하는 것만 구현한다.

## 2. 초기 구현 범위

- 자료명 입력
- 파일 또는 폴더 URL 입력
- 간단한 자료 종류 선택
- 관련 업무·사업·행사 연결
- 직원 공개 여부 선택
- 중요자료 또는 최종자료 표시
- 등록자와 등록일 기록
- 자료 열기 버튼
- 자료 숨김 또는 보관

## 3. 초기 화면 예시

```text
관련 자료
- 작업 매뉴얼  [자료 보기]
- 제품 사진    [사진 보기]
- 행사 계획서  [자료 보기]
```

일반 근로자는 본인 업무에 연결된 쉬운 핵심자료만 읽기용으로 본다. 긴 PDF 원문과 관리자용 자료는 일반 근로자 화면에 직접 노출하지 않는다. 팀장과 관리자는 링크를 직접 등록한다.

## 4. 초기 버전에서 제외

다음 기능은 플랫폼 핵심 업무 기능을 늦출 수 있으므로 후속 단계로 보류한다.

- Google 로그인 연동
- Drive API
- Google Picker
- 플랫폼 직접 파일 업로드
- 자동 폴더 생성
- 파일 내용 검색
- 문서 버전 비교
- 복잡한 자료별 권한 설정
- 자동 최종본 전환
- Calendar·Gmail 연동
- 사진·영상 원본 관리 자동화
- 계약관리 전용 시스템

## 5. 단순 권한 원칙

초기에는 공개범위를 복잡하게 나누지 않고 다음 두 단계만 사용한다.

- 업무 참여자에게 공개
- 팀장·관리자만 공개

근로자지원·인사·계약 등 민감자료는 초기 플랫폼 링크 등록 대상에서 제외하거나 관리자 전용으로만 관리한다.

## 6. 개발 판단

이 범위는 기본 입력 폼, URL 저장, 목록 표시, 역할별 노출 제어만 필요하므로 초기 버전에서 구현 가능하다. 기존 업무·사업 데이터에 자료 링크 테이블 또는 필드를 연결하는 방식으로 개발하며, Google Workspace 연동 때문에 전체 플랫폼 개발일정이 늦어지지 않도록 한다.
