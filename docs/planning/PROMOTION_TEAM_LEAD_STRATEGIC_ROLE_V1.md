# 홍보팀장 전략·사업기획 역할 v1

상태: **확정**

이 문서는 태장 홍보팀장이 생산팀장·물류팀장과 다른 성격의 역할을 수행한다는 점과, 대표이사가 만들어 가는 외부사업을 기획·실행·확장하는 책임을 정의합니다.

## 1. 역할 성격

- 생산팀장과 물류팀장은 이미 정해진 업무를 안정적으로 배정·관리·완료하는 운영형 팀장입니다.
- 홍보팀장은 기존 홍보업무를 운영하는 동시에 새 사업과 대외활동을 기획하는 전략·사업개발형 팀장입니다.
- 대표이사가 발굴하거나 추진하는 외부사업, 협력, 행사, 사회공헌, ESG, 교육, 콘텐츠 기회를 실제 실행계획으로 구체화합니다.
- 단순 게시물 관리자나 콘텐츠 검수자에 머물지 않습니다.

## 2. 홍보팀장의 두 가지 업무축

### 기존 운영업무

- 홈페이지와 SNS 채널 운영
- 콘텐츠 일정과 촬영 관리
- 직원 업무배정과 진행확인
- 콘텐츠 검토·승인·게시
- 사진·영상·디자인 자산 관리
- 외부 콘텐츠의 홈페이지 기록
- 홍보성과와 누락 현황 관리

### 신규 기획업무

- 대표이사가 제안한 신규 사업·활동의 목적과 방향 정리
- 대상기관·협력사·참여자·일정·예산·필요자료 검토
- 사업 또는 행사의 실행계획 초안 작성
- 홍보·콘텐츠·대외협력 관점의 활용방안 설계
- 생산·물류·근로자지원·영업 등 관련 부서와 협의
- 대표이사와 운영총괄에게 검토안·선택지·위험요소 보고
- 승인된 사업을 세부 업무로 나누어 담당자에게 배정
- 진행상황·성과·후속과제를 기록하고 다음 사업으로 연결

## 3. 대표이사와의 관계

- 대표이사는 새로운 사업기회와 큰 방향을 제시합니다.
- 홍보팀장은 이를 조사·정리하여 실행 가능한 기획안으로 만듭니다.
- 운영총괄은 부서 간 자원·일정·승인 필요사항을 조정합니다.
- 최종 사업방향과 중요한 대외결정은 대표이사가 승인합니다.

기본 흐름:

```text
대표이사 아이디어·외부사업 제안
→ 홍보팀장 조사·기획안 작성
→ 관련 부서 사전협의
→ 운영총괄 검토·조정
→ 대표이사 결정
→ 홍보팀장 실행계획·업무배정
→ 진행관리·대외홍보·공식기록
→ 결과보고·후속사업 제안
```

## 4. 홍보팀장 대시보드 추가영역

공통 팀장 대시보드 위에 다음 영역을 추가합니다.

- 대표이사 신규 요청
- 기획 중 사업
- 검토가 필요한 아이디어
- 관련 기관·협력사 협의 현황
- 부서 간 사전협의
- 운영총괄 검토 대기
- 대표이사 결정 대기
- 승인된 신규사업 실행현황
- 일정·예산·자료·담당자 준비상태
- 종료 사업의 결과·후속과제

첫 화면은 기존 콘텐츠 운영과 신규사업 기획을 구분하여 보여 줍니다.

## 5. 신규사업 상태

- 아이디어 접수
- 기초 조사
- 기획 중
- 부서 협의 중
- 운영총괄 검토
- 대표이사 결정 대기
- 승인
- 실행 준비
- 진행 중
- 완료
- 보류
- 종료·기록

## 6. 권한과 제한

홍보팀장은 다음을 할 수 있습니다.

- 신규사업 기획안과 선택지 작성
- 관련 부서 협의 요청
- 자료수집과 사전조사 업무배정
- 승인 후 실행업무 배정과 진행관리
- 홍보·대외협력·기록 계획 수립

다음 사항은 단독 확정하지 않습니다.

- 회사의 신규사업 최종 착수
- 중요한 계약·단가·예산
- 인력의 공식 부서·직무 변경
- 대표이사 명의의 대외 약속
- 회사의 중대한 공식 입장

## 7. 개발 검수 기준

- 홍보팀장 화면이 단순 콘텐츠 게시관리로 제한되지 않는가
- 기존 운영업무와 신규 기획업무가 구분되는가
- 대표이사 요청이 조사·기획·협의·결정·실행 흐름으로 관리되는가
- 승인 전 아이디어가 직원 실행업무로 잘못 배정되지 않는가
- 홍보팀장이 관련 부서와 협의하고 운영총괄 조정을 요청할 수 있는가
- 완료 사업의 결과와 후속과제가 공식 기록으로 남는가
