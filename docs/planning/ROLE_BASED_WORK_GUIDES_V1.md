# 직책별 업무 가이드 제공 원칙 v1

상태: **확정**

## 1. 목적

태장 업무 플랫폼의 매뉴얼은 별도 문서로만 보관하지 않고, 사용자가 로그인한 뒤 자신의 직책과 현재 업무에 맞는 내용을 업무 화면에서 바로 확인할 수 있도록 제공합니다.

## 2. 기본 원칙

- 공통 기준은 하나의 공식 매뉴얼 원본으로 관리합니다.
- 로그인한 사용자의 직책·권한·소속 모듈에 따라 필요한 범위만 보여줍니다.
- 긴 전체 매뉴얼을 강제로 읽게 하지 않습니다.
- 현재 작업 화면과 관련된 도움말을 즉시 열 수 있어야 합니다.
- 가이드 내용은 실제 회사 업무 기준과 플랫폼 조작 방법을 함께 설명합니다.
- 기획 변경 시 공식 원본과 화면별 도움말이 함께 수정돼야 합니다.

## 3. 역할별 제공 범위

### 홍보직원

- 오늘 해야 할 일
- 글 작성 방법
- 사진·영상 정리 방법
- 블로그 원고 작성 기준
- 인스타그램·X 파생 작성법
- 승인 요청과 보완 필요 처리
- 개인정보·인물 사진 주의사항
- 팀장에게 도움 요청하는 방법

### 홍보팀장

- 팀 업무와 최소 발행량 관리
- 승인 요청 검토
- 직접 수정 가능 범위
- 보완 요청 기준
- 운영총괄 전달 기준
- 글감 가이드 작성
- 사진·영상 자료 점검
- 유튜브 기획안 작성
- 홈페이지 점검과 업무 이월

### 운영총괄

- 중요 콘텐츠 검토 기준
- 대표이사 확인 필요 여부 판단
- 모회사·고용·인증·금액·보도자료 확인
- 기획안 승인
- 장기 미승인 업무 관리
- 채널 운영 현황과 주요 누락 확인
- 공개 중단·수정·재승인 절차

### 대표이사

- 확인이 필요한 이유
- 핵심 내용 요약
- 운영총괄 검토 의견
- 승인·보완 의견·게시 보류

## 4. 화면 배치

모든 역할의 대시보드에는 `업무 가이드` 진입점을 둡니다.

각 화면에는 상황별 도움말을 제공합니다.

- 글쓰기 화면: 글 작성 가이드
- 승인 화면: 직접 수정과 보완 요청 기준
- 사진·영상 화면: 자료 분류 기준
- 채널 현황 화면: 최소 발행 기준과 예외 처리
- 운영총괄 승인 화면: 중요 콘텐츠 확인 기준

도움말은 현재 업무를 방해하지 않도록 별도 패널이나 안내창으로 열립니다.

## 5. 향후 확장

이 원칙은 홍보 모듈뿐 아니라 물류·재고·생산·교육·영업 등 후속 모듈에도 동일하게 적용합니다.

## 6. 개발 검수 기준

- 로그인한 역할에 맞는 가이드만 우선 노출되는가
- 사용자가 현재 업무 화면에서 관련 도움말을 바로 열 수 있는가
- 업무 기준과 시스템 사용법이 서로 분리되지 않고 연결돼 있는가
- 하나의 공식 원본을 기준으로 내용이 관리되는가
- 권한이 없는 업무의 상세 운영 기준이 불필요하게 노출되지 않는가

## 7. 확정 문장

> 태장 업무 플랫폼의 매뉴얼은 하나의 공식 기준으로 관리하되, 사용자가 로그인하면 직책과 권한에 맞는 업무 가이드가 표시되고 각 업무 화면에서 필요한 안내를 즉시 확인할 수 있도록 제공한다.
