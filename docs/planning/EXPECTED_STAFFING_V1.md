# 태장 1차 목표 인원 구성 v1

상태: **확정**

이 문서는 태장의 1차 최종 고용 규모와 조직·대시보드·계정 설계에 적용할 예상 인원 구성을 기록합니다.

## 1. 경영진과 책임자

- 대표이사: 이영희
- 총괄 등기이사: 김형철
- 비등기이사·영업담당: 이선영
- 물류팀장: 황성하
- 홍보팀장: 이환호

## 2. 지원·지도 인력

### 근로자지원팀

- 근로자지원 담당 1명
- 현재 수습 상태
- 팀장 공석
- 주요 검토와 승인은 운영총괄이 수행

### 민화 생산 지도

- 민화 생산 지도 작가 1명
- 주 2회 풀타임 지도
- 생산팀의 민화 작업과 작품 완성도 지도를 담당
- 직원 인사·상담·장애정보에는 접근하지 않는 외부 지도자 또는 별도 계약 인력으로 구분

### 안전교육 및 환경정비 지도

- 안전교육 담당 선생님 1명
- 주 1회 풀타임 지도
- 월 1~2회 진행되는 환경정비 사업을 대비한 안전교육과 작업지도를 담당
- 직원 인사정보에는 접근하지 않고 교육 일정·참여·교육내용·안전 특이사항만 제한적으로 관리

## 3. 장애인 근로자

- 1차 최종 채용 목표: 장애인 근로자 25명
- 모든 재직 장애인 근로자에게 개인 모바일 계정을 기본 제공
- 부서·오전반·오후반 배치는 별도 인원 배치표에서 관리
- 일반 근로자 화면은 본인 업무, 공지, 일정, 교육, 지원 요청 중심으로 구성
- 장애 유형과 정도는 조직도나 일반 대시보드에 표시하지 않음

## 4. 농장지사

- 현재 인원 없음
- 지사장과 농장 근로자 모두 공석
- 조직도에는 운영 예정 조직으로 유지
- 별도 계정과 지사 대시보드는 인력 배치 전까지 활성화하지 않음

## 5. 인원 집계 기준

### 내부 임직원·근로자

- 경영진·팀장·지원담당: 6명
  - 대표이사 1명
  - 총괄 등기이사 1명
  - 비등기이사 1명
  - 물류팀장 1명
  - 홍보팀장 1명
  - 근로자지원 담당 1명
- 장애인 근로자: 25명
- 내부 인원 합계: **31명**

### 외부 또는 별도 계약 지도 인력

- 민화 생산 지도 작가 1명
- 안전교육 담당 선생님 1명
- 지도 인력 합계: **2명**

### 전체 운영 참여 인원

- 내부 인원 31명
- 외부·별도 계약 지도 인력 2명
- 전체 운영 참여 인원: **33명**

민화작가와 안전교육 담당자의 법적 고용형태가 직원인지 외부 계약자인지에 따라 공식 직원 수에는 포함하지 않을 수 있습니다. 플랫폼에서는 `재직 직원`, `외부 지도자`, `공석`을 분리해 관리합니다.

## 6. 플랫폼 설계 반영

- 내부 임직원과 근로자는 개인 계정을 원칙으로 함
- 외부 지도자는 필요 시 제한 계정을 부여하거나 내부 담당자가 대신 기록
- 대표이사와 운영총괄은 전사 접근이 가능하되 기본 대시보드는 승인·위험·예외 중심으로 압축
- 팀장은 부서 관리화면, 일반 근로자는 단순 모바일 화면 사용
- 공석 조직에는 계정과 대시보드를 생성하지 않음
- 퇴사·계약 종료 시 계정 정지, 활성 세션 종료와 접근권한 회수를 즉시 수행

## 7. 후속 확정 필요사항

- 장애인 근로자 25명의 부서별 최종 배치
- 오전반·오후반별 최종 인원
- 민화작가와 안전교육 담당자의 계약·고용 형태
- 생산팀 임시 업무관리자 지정
- 홍보팀 장애인 근로자 배치
- 물류팀 장애인 근로자 외 나머지 인원의 배치
