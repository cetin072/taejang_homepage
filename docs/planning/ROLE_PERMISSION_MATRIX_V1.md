# 태장 업무플랫폼 역할별 권한표 v1

상태: **확정**

## 1. 원칙

- 권한은 직함만이 아니라 실제 담당업무에 따라 배정한다.
- 회원가입 직후에는 `승인 대기` 상태이며 내부자료에 접근할 수 없다.
- 시스템 최고관리자 김형철이 가입상황을 확인한 뒤 부서·직책·역할·근무반·지원화면을 배정한다.
- 버튼을 숨기는 것만으로 권한을 구현하지 않고 데이터 접근단계에서도 제한한다.
- 대표이사와 운영총괄의 전사 접근은 허용하되 민감정보 열람은 업무상 필요성과 감사기록을 적용한다.
- 외부지도자와 일반 근로자에게 불필요한 인사·상담·계약·전사정보를 노출하지 않는다.

## 2. 역할 정의

### 대표이사

- 경영 최종결정
- 운영총괄이 상신한 중요안건 승인
- 필요 시 전사 업무·승인·기록 열람
- 일상 계정관리와 일반 실무승인은 기본 담당이 아님

### 운영총괄

- 회사 운영 전반 총괄
- 일반 업무의 최종 승인
- 부서 간 조정
- 중요 위험·지연 관리
- 대표이사 상신 판단
- 필요 시 전사 업무와 기록 열람

### 시스템 최고관리자

- 초기 확정 담당자: 김형철
- 가입 승인·거절
- 계정 상태 관리
- 부서·직책·역할·근무반 배정
- 관리자 대리처리 설정
- 마지막 최고관리자 보호
- 시스템 설정과 중요 감사기록 확인

운영총괄과 시스템 최고관리자는 동일 인물이 겸임할 수 있으나 권한 개념은 분리한다.

### 부서 팀장

- 본인 부서 업무 작성·배정·검토
- 부서 직원·작업반 현황 확인
- 업무 보완 요청
- 부서 공지·일정·작업방법 관리
- 부서 간 요청 수락·조정·담당자 배정
- 민감 상담원문·전사 계정상태 변경 불가

### 총괄반장·업무반장·현장 관리자

- 지정 작업반의 오늘 업무와 작업방법 확인
- 작업반 배정·현장 확인
- 완료·예외·문제·인계 최소기록
- 작업방법 작성·수정
- 근로자지원 연결
- 인사결정·상담원문 열람·계정관리 불가

### 근로자지원 책임자

- 제한된 상담·지원 기록 열람·검토
- 담당자 초안 보완 요청·확정
- 지원계획·조치·후속확인 관리
- 필요한 현장 협조사항 분리 전달
- 공식 인사·계약·급여·징계·해고 최종결정 불가

### 근로자지원 담당자

- 상담·관찰·지원 요청 기록
- 지원계획 초안
- 현장·보호자·기관 협의 기록
- 후속확인
- 책임자 검토 요청
- 다른 직원의 불필요한 상담기록 열람 불가

### 홍보팀장

- 홍보업무 배정
- 직원 작성내용 검토·보완
- 콘텐츠 일정·자료·수정이력 관리
- 운영총괄 승인 요청
- 일반 계정관리·상담원문 열람 불가

### 홍보 직원

- 배정된 홍보업무 확인
- 콘텐츠 초안·자료·결과 기록
- 정형화된 작성도우미 사용
- 팀장 검토 요청
- 최종 승인·게시권한은 별도 부여 시에만 가능

### 일반 사무직원

- 본인·소속부서 업무 확인
- 일반 행정·협업 기록 작성
- 관련 자료 링크 연결
- 팀장 검토 요청
- 타 부서 직접 직원배정·최종승인·계정관리 불가

### 일반 장애인 근로자

- 오늘의 업무 열람
- 작업방법 열람
- 내 일정·근무시간 열람
- 중요공지 열람 및 필요한 공지만 확인
- 자주 보는 안내 열람
- 시작·완료·실적·출퇴근·자유보고·상담기록 입력 없음

### 근로지원인

- 지정 근로자의 업무수행 지원
- 필요한 작업안내와 일정 열람
- 현장에서 확인한 지원 필요사항을 담당자에게 전달
- 상담원문·전체 인사정보·계정관리 불가

### 외부지도자

- 본인 일정
- 본인 지도업무
- 준비사항
- 필요한 작업자료
- 간단한 결과·전달사항

내부 인사·급여·계약·전체 직원목록·상담기록에는 접근할 수 없다.

## 4. 데이터 범위

### 대표이사·운영총괄

- 필요 시 전사 범위
- 기본 화면은 압축된 중요정보만 표시
- 민감정보 열람 시 감사기록

### 부서 팀장

- 소속부서 직원·업무·공지·일정·관련자료
- 부서 간 요청 중 본인 부서가 요청·수신한 건
- 다른 부서의 세부 인사·상담정보는 접근 불가

### 현장 책임자

- 본인 담당 작업반
- 담당 작업과 작업방법
- 필요한 직원별 업무지원 정보
- 상담원문과 진단정보 접근 불가

### 근로자지원

- 담당 근로자의 제한된 지원·상담기록
- 현장 업무정보 중 지원에 필요한 범위
- 급여·계약·징계 등은 제안·협의만 가능

### 일반 장애인 근로자

- 본인 업무·근무시간·일정·공지
- 본인에게 지정된 작업방법·생활안내
- 다른 직원의 업무·개인정보·기록 접근 불가

### 공용 태블릿

- 작업반 공통 업무·작업방법·중요공지
- 개인 일정·이름·지원정보·상담정보 표시 금지

## 5. 권한 변경

- 같은 계정에서 부서·직책·역할을 변경한다.
- 운영총괄이나 대표이사가 계정관리를 수행하려면 해당 계정에 `super_admin` 역할을 별도로 부여해야 한다.
- 역할 변경 전 기존 담당업무와 승인대기 건을 확인한다.
- 퇴사·정지 시 새 업무 배정과 작성권한을 즉시 차단한다.
- 담당 콘텐츠·업무·지원 건을 재배정한다.
- 역할·권한 변경은 감사기록을 남긴다.

## 6. 초기 권한 검수 기준

- 승인 대기 사용자가 내부자료를 볼 수 없는가
- 일반 근로자가 본인 대상 정보만 읽을 수 있는가
- 부서 팀장이 타 부서 민감자료를 볼 수 없는가
- 현장 책임자가 상담원문을 볼 수 없는가
- 근로자지원 담당자가 공식 인사결정을 내릴 수 없는가
- 외부지도자가 내부 직원목록과 인사자료를 볼 수 없는가
- 마지막 최고관리자를 제거할 수 없는가
- 계정 정지·퇴사 후 기존 세션과 작성권한이 차단되는가
- 화면 표시와 데이터 접근권한이 일치하는가
