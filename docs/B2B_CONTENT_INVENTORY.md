# B2B 공개 콘텐츠 인벤토리

이 문서는 기업·기관 대상 공개 콘텐츠의 현재 상태와 회사에서 추가로 준비해야 할 자료를 정리합니다. 계약 조건, 가격, 수익 구조, 비공개 고객사 정보는 이 문서의 관리 대상이 아닙니다.

## 현재 검토 상태

PR #12의 `partnership.html`과 `resources.html`은 구조와 기능 기반만 임시 승인되었습니다. 사용자는 두 페이지의 전반적인 콘텐츠·표현·정보 구성에 만족하지 않았으므로, 현재 버전을 최종 B2B 콘텐츠로 간주하지 않습니다.

- 기능적 기반: 유지
- 콘텐츠 문구·정보 구성: 후속 재검토 필요
- 실제 사진·회사소개서·인증 공개본·협력 사례 수령 후 재설계
- 이번 단계에서는 페이지 코드를 추가 수정하지 않음

## 현재 공개 콘텐츠

| 항목 | 공개 위치 | 상태 | 비고 |
| --- | --- | --- | --- |
| 회사 정체성·인증번호 | 메인 홈페이지 | 공개 중 | 최신성은 공식 자료로 재확인 |
| 사업 상태 | 메인 홈페이지, 기업 협력 상세 | 공개 중 | 현재 주력·협력 운영·준비 중·협의형 구분 |
| 기업 협력 분야·일반 절차 | `partnership.html` | 구조 임시 승인·콘텐츠 재검토 | 개별 계약 절차나 혜택을 확정적으로 말하지 않음 |
| 기업 협력 FAQ | `partnership.html#partnership-faq` | 기능 기반 임시 승인 | 질문·답변 구성은 실제 자료 수령 후 재검토 |
| 일터·활동 기록 | `workplace.html`, `activities.html` | 공개 중 | 사진과 시제는 승인·사실 확인 후 갱신 |
| 문의·오시는 길 | 메인 홈페이지 | 공개 중 | 전화·이메일·주소 최신성 확인 필요 |
| 자료실 구조 | `resources.html` | 구조 임시 승인·콘텐츠 재검토 | 실제 파일 없이 상태만 안내하며, 실제 자료 수령 후 재설계 |

## 자료 수령 대기

- 공개용 회사소개서 PDF
- 개인정보를 제거한 표준사업장 인증 관련 공개 자료
- 승인된 민화·포장·검수·사업장 사진
- 기업·지역 협력 사례와 기업명·로고·인물 공개 승인
- 공식 보도자료와 출처
- 실제 채용이 확정된 경우의 공개용 채용 자료

## 회사에서 준비할 자료

1. 자료의 최신 버전과 공개 가능 여부
2. 회사소개서·인증 자료의 공개 승인자와 확인일
3. 사진별 촬영일, 촬영자, 인물·기업 로고 공개 승인 상태
4. 협력 사례의 공개 범위와 확인 가능한 사실
5. 보도자료의 공식 원문, 날짜, 외부 링크 사용 승인
6. 채용 공고가 필요한 경우의 직무·일정·연락 방법과 개인정보 처리 기준

## 공개 승인 필요 자료

| 자료 | 확인할 사항 |
| --- | --- |
| 기업명·로고 | 기업의 외부 공개 승인과 최신 로고 여부 |
| 협력 사례 | 사실관계, 계약·수량·가격 비공개, 공개 범위 |
| 활동 사진 | 인물 동의, 문서·주소·송장·고객 정보 노출 여부 |
| 인증·회사소개서 PDF | 개인정보 제거, 버전, 공개 승인 |
| 보도자료 | 공식 배포본·날짜·출처 |

## 향후 관리자 시스템 관리 대상

- 회사소개서·인증 자료·보도자료의 버전과 공개 상태
- 기업 협력 FAQ와 사례
- 사진·PDF의 승인 상태와 대체 이력
- 채용·배너·팝업 등 실제 운영 콘텐츠
- 문의 접수와 담당자 배정·회신 기록
