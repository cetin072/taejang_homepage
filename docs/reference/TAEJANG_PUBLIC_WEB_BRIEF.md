# 태장 공개 홈페이지 콘텐츠 기준

## 문서 목적과 적용 범위

이 문서는 농업회사법인 태장 주식회사의 공개 홈페이지와 공개 콘텐츠에서 사용할 정보·표현의 기준입니다. 계약서, 정관, 주주명부, 투자·협약 자료의 원문을 복사하거나 대체하는 문서가 아닙니다.

홈페이지 문구, 공지, 활동 소개, 사업 소개, 기업 협력 안내, 사진·로고 설명을 작성하거나 수정할 때 이 문서를 먼저 확인합니다. 사실관계·공개 승인·사업 상태가 불명확하면 임의로 확정하지 않고 사용자 확인을 요청합니다.

## 홈페이지의 역할

홈페이지는 회사소개서나 투자·협약 제안서가 아닙니다. 공개 홈페이지는 다음 역할에 집중합니다.

1. 태장의 공식 정체성과 신뢰를 형성합니다.
2. 실제 사업과 준비 사업을 구분해 소개합니다.
3. 장애인 근로자의 일터와 활동을 존중하는 방식으로 소개합니다.
4. 기업·기관과의 협력 가능성을 안내합니다.
5. 적절한 문의로 연결합니다.

신규 모회사 모집을 홈페이지 전면에서 강조하지 않습니다. `다섯 번째 모회사 모집`, `YOUR COMPANY?` 같은 문구는 공개 메인 홈페이지의 기본 문구로 사용하지 않습니다.

## 주요 방문자와 공개 톤

- 기존 모회사와 협력기업
- 신규 기업·기관 담당자
- 공공기관과 지역사회
- 근로자와 가족
- 일반 방문자

B2B 방문자는 중요한 대상이지만, 홈페이지가 투자·출자 모집 사이트처럼 보이지 않도록 합니다. 홈페이지는 기본 정보와 협력 가능성을 명확히 보여 주고, 구체적인 제도·참여 구조·조건은 상담과 별도 자료에서 다룹니다.

## 사업 영역과 상태

| 사업 영역 | 상태 | 공개 설명 | 공개 시 주의사항 |
| --- | --- | --- | --- |
| 민화·문화 굿즈 | 현재 주력 | 민화 작업을 바탕으로 달력, 카드, 기념품 등 기업·기관용 문화 굿즈를 제작합니다. | 실제 제작 범위와 승인된 사례만 사용합니다. |
| 기업 연계 환경활동 | 협력 운영 | 기업 및 지역사회와 협력하여 환경정비와 사회공헌 활동을 진행합니다. | 참여 기관·활동명은 공개 승인과 사실 확인 후 사용합니다. |
| 농식품 상품 개발 | 준비 중 | 농업 기반을 활용한 농식품 상품 개발과 제조 사업을 준비하고 있습니다. | 혼합차·다류 제조는 현재 준비 중입니다. `제조합니다`, `생산합니다` 등 완료형 표현을 사용하지 않습니다. |
| 기업 물품·업무 협력 | 협의형 | 기업에 필요한 물품과 반복 업무를 검토하여 근로자에게 적합한 방식으로 협력합니다. | 구매대행의 수익 구조, 단가, 마진, 정산 방식은 공개하지 않습니다. |

## 기업 협력 표현

### 권장 표현

- 기업과 함께 지속 가능한 일자리를 만듭니다.
- 물품, 업무, 문화 콘텐츠, 지역사회 활동 등 다양한 협력을 논의할 수 있습니다.
- 기업의 상황과 필요에 맞는 협력 방식을 함께 검토합니다.
- 자회사형 장애인 표준사업장 제도와 관련된 사항은 상담을 통해 안내합니다.

### 전면 사용을 피할 표현

- 부담금 완전 해소
- 막대한 부담금 절감
- 고용 실적 자동 인정
- 참여하지 않을 이유가 없습니다
- 다섯 번째 모회사 모집
- 지분 참여 시 즉시 혜택 발생

고용 인정, 부담금 절감, ESG 효과는 법적 요건과 기업별 참여 구조에 따라 달라질 수 있습니다. 홈페이지에서는 이를 단정적으로 표현하지 않고, 상담과 공식 검토가 필요한 사항으로 안내합니다.

## 공개 수준 분류

| 등급 | 정의 | 예시 |
| --- | --- | --- |
| `PUBLIC` | 홈페이지에 직접 공개할 수 있는 정보 | 법인명, 인증번호, 사업 영역, 운영 원칙, 공개 승인된 활동, 공식 연락처 |
| `LIMITED` | 개념과 요약만 공개하고 세부사항은 상담 후 제공하는 정보 | 자회사형 표준사업장 구조, 고용 인정 가능성, ESG 연계 효과, 기업 협력 절차 |
| `INTERNAL` | 홈페이지와 공개 콘텐츠에 사용하면 안 되는 정보 | 주주명부, 지분율, 계약대금, 보장재원, 부담금 절감 금액, 원가·마진, 인정 인원, 협상 중인 기업, 계약서 원문, 내부 일정 |
| `VERIFY` | 공식 서류 또는 대표 확인 후에만 사용할 수 있는 정보 | 최신 주소, 사업 인허가, 준비 사업의 개시 여부, 협력사 명칭과 로고, 인물 사진 사용 동의 |

## 문구 작성 원칙

홈페이지 문구는 항상 다음 질문을 통과해야 합니다.

1. 공식 자료나 실제 운영으로 확인된 사실인가?
2. 현재형으로 말해도 되는가?
3. 공개할 필요가 있는 정보인가?
4. 외부인이 과장·투자 모집·부담금 영업으로 오해할 가능성이 없는가?
5. 장애인을 시혜적 대상으로 묘사하지 않는가?

준비 사업에는 다음 상태 표현을 사용합니다.

- 준비 중
- 단계적으로 확대
- 협의 중
- 향후 연계 예정

미확정 사업에는 다음 표현을 사용하지 않습니다.

- 운영하고 있습니다
- 생산하고 있습니다
- 제공하고 있습니다
- 보장합니다

## 회사소개서와 홈페이지의 구분

| 구분 | 다루는 내용 |
| --- | --- |
| 홈페이지 | 기본 정보, 신뢰 형성, 사업 개요, 일터와 활동, 협력 가능성, 문의 연결 |
| 회사소개서 | 제도 구조, 지분·협약 방식, 고용 실적 산입 구조, 부담금 절감 가능성, 참여 절차, 사업모델과 운영 구조 |
| 계약·협의자료 | 지분율, 인정 인원, 계약대금, 발주 규모, 정산 방식, 계약 조건 |

홈페이지는 공개 가능한 기본 정보 중심으로 작성하고, 상세 협의·재무·계약 조건은 회사소개서 또는 계약·협의자료에서만 다룹니다.

## Codex 작업 시 준수사항

앞으로 홈페이지 문구를 추가하거나 수정할 때는 반드시 이 문서를 먼저 읽습니다. 다음 상황에서는 임의로 판단하지 말고 사용자 확인을 요청합니다.

- 현재 사업인지 준비 사업인지 불명확한 경우
- 외부 공개 승인 여부가 불명확한 경우
- 모회사·기관명 또는 로고를 사용하는 경우
- 법적·재무적 혜택을 표현하는 경우
- 수치·주소·인증 정보가 문서마다 다른 경우
- 인물이 포함된 사진을 사용하는 경우

## 사진 파일명 규칙

공개 홈페이지용 사진은 `taejang-[주제]-[세부용도]-[번호].jpg` 형식을 사용합니다. 영문 소문자와 하이픈을 사용하고, 한글 파일명·공백·괄호는 피합니다.

예: `taejang-palyong-workplace-01.jpg`, `taejang-minhwa-work-01.jpg`, `taejang-environment-activity-01.jpg`, `taejang-packing-work-01.jpg`, `taejang-farm-overview-01.jpg`

## 공개 전 최종 확인

- `PUBLIC`, `LIMITED`, `INTERNAL`, `VERIFY` 등급을 확인합니다.
- 개인정보, 주주별 지분, 계약금액, 내부 일정, 협상 정보가 포함되지 않았는지 확인합니다.
- 준비 사업은 현재 사업처럼 표현하지 않았는지 확인합니다.
- 사진·로고·협력사명은 사용 승인과 최신성을 확인합니다.
