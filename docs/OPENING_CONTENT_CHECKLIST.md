# 신규 사업장 개소 콘텐츠 체크리스트

기준 예정일: **2026.08.12**

이 문서는 실제 개소 확인 전후에 홈페이지 문구와 사진을 일관되게 관리하기 위한 운영용 체크리스트입니다. 이 문서를 작성하는 것만으로 현재 홈페이지의 예정 문구를 완료형으로 바꾸지 않습니다.

## 개소 전 사용 문구

- 태장 신규 사업장 개소식
- 태장의 새로운 사업장이 문을 엽니다
- 신규 사업장을 열 예정입니다
- 새로운 업무 공간을 준비하고 있습니다
- 개소 예정일: 2026.08.12

개소 전에는 `개소했습니다`, `운영을 시작했습니다`, `행사를 마쳤습니다`처럼 완료를 뜻하는 표현을 사용하지 않습니다.

## 개소 후 변경 문구

실제 개소와 공개 승인이 확인된 뒤에만 아래처럼 상태를 바꿉니다.

- 신규 사업장이 문을 열었습니다
- 개소식을 진행했습니다
- 새 업무 공간에서 [확인된 업무]를 시작했습니다

완료형 문구는 실제 날짜·업무 시작 상태·공개 범위를 확인한 뒤 사용합니다. 확인되지 않은 참석 규모, 협력 실적, 운영 성과는 추가하지 않습니다.

## 개소 후 확인사항

- 실제 개소 날짜가 2026.08.12와 같은지 확인
- 실제 업무 시작 여부와 공개 가능한 업무 범위를 확인
- `office-1.jpg` 또는 승인된 개소식 사진의 공개 승인 여부 확인
- 참석자·근로자·방문객의 인물 공개 동의 확인
- 기업명·로고·현수막·문서가 공개 가능한지 확인
- 초대장 링크를 유지할지, 기록으로 남길지, 제거할지 결정
- `예정`, `준비 중`, `열 예정` 등 예정 표현을 해당 위치에서 모두 점검
- 메인 미리보기, 상세 글, 연혁의 날짜와 시제를 통일
- Preview에서 제목·사진·모바일 줄바꿈을 확인

## 사진 공개 승인 확인

사진을 적용하기 전에는 [사진 운영 가이드](PHOTO_GUIDE.md)의 개인정보·보안 기준과 교체 절차를 따릅니다. 승인되지 않은 인물·기업명·로고가 보이는 사진은 홈페이지나 공개 저장소에 올리지 않습니다.
