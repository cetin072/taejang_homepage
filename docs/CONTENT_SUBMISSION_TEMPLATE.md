# 태장 홈페이지 콘텐츠 작성 양식

아래 항목을 작성해 콘텐츠 담당자와 내부 확인자에게 전달합니다. 사실·날짜·사진 공개 승인 상태가 불명확하면 게시 요청 전에 확인합니다.

## 작성 양식

- 글 구분: `일터 이야기` / `태장의 활동`
- 카테고리:
- 작성일: `YYYY.MM.DD`
- 제목:
- 한 줄 요약:
- 도입문:
- 중간 제목 1:
- 본문 1:
- 중간 제목 2:
- 본문 2:
- 사진 파일명:
- 사진 설명:
- 인물 공개 동의 여부: `확인됨` / `확인 필요` / `해당 없음`
- 기업명·로고 공개 승인 여부: `확인됨` / `확인 필요` / `해당 없음`
- 공개 희망일:
- 작성자:
- 내부 확인자:

## 일반적인 작성 예시

- 글 구분: 일터 이야기
- 카테고리: 일터 운영
- 작성일: 2026.08.20
- 제목: 작업 순서를 함께 점검합니다
- 한 줄 요약: 작업대와 자재 위치를 살피며 이해하기 쉬운 업무 흐름을 정리했습니다.
- 도입문: 근로자가 작업을 익히기 쉽도록 작업 순서와 공간을 함께 점검했습니다.
- 중간 제목 1: 작업 전 준비
- 본문 1: 필요한 도구와 자재를 가까운 위치에 정리하고, 작업에 방해가 되는 요소를 확인했습니다.
- 중간 제목 2: 계속 확인하는 업무 흐름
- 본문 2: 현장에서 발견한 불편은 기록하고, 근로자의 작업 속도를 고려해 개선 방법을 검토합니다.
- 사진 파일명: office-2.jpg
- 사진 설명: 개인정보와 문서가 보이지 않는 정돈된 작업공간
- 인물 공개 동의 여부: 해당 없음
- 기업명·로고 공개 승인 여부: 해당 없음
- 공개 희망일: 2026.08.22
- 작성자: 콘텐츠 담당자
- 내부 확인자: 운영 확인자

예시는 일반적인 형식 안내입니다. 실제 운영 사실, 성과, 협력사명, 인물 사진은 확인된 내용으로만 교체합니다.

제출 전에는 [사진 운영 가이드](PHOTO_GUIDE.md)를 확인하고, 반영 전에는 [홈페이지 공개 전 최종 검수 체크리스트](PUBLISH_CHECKLIST.md)를 사용합니다.
