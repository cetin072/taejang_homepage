# Brand assets

## Taejang CI guide

- `taejang-ci-guide-public.pdf`: Public-safe edition containing pages 1-7 of the original CI guide.
- Personal mobile numbers, personal email addresses, and business-card/contact layouts from pages 8-9 are excluded.
- The original full guide must remain in internal company storage and must not be committed to the public repository.

## Partner logos

- `assets/images/partners/bumhan.svg`: Web vector converted from the supplied official AI file.
- `assets/images/partners/bumhan.png`: Raster fallback generated from the same source.
- `assets/images/partners/samhyun.jpg`: Supplied high-resolution horizontal logo. Replace with official SVG/AI when available.
- `assets/images/partners/cheungwoo-bj.png`: Cropped web copy from the supplied JPEG. Replace with official vector or transparent high-resolution PNG when available.

## Not committed

- Low-resolution duplicate `삼현.png` was excluded.
- The full Hyundai BNG Steel legacy CI archive was excluded because it contains multiple 2011-era source files and the currently approved logo variant has not been confirmed.
- Original AI/ZIP source files should be retained in restricted internal storage rather than the public website repository.

## Publication control

Possession of a logo file does not by itself prove publication approval. Partner logos should be displayed publicly only after the company confirms that the selected version and use are authorized.
