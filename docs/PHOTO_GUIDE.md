# 태장 홈페이지 사진 운영 가이드

## 1. 촬영 목적

이 가이드는 태장 홈페이지의 사진자료 안내박스를 실제 승인 사진으로 교체할 때 사용합니다. 사진은 일터와 활동을 사실에 맞게 소개하기 위한 자료이며, 촬영·공개 승인 전에는 홈페이지에 연결하지 않습니다.

현재 안내박스에 표시된 파일명은 **촬영 예정 사진의 통제 파일명**입니다. 현재 `images/`에 있는 승인 이미지의 파일명과 경로는 바꾸지 않습니다.

## 2. 파일명 목록

| 파일명 | 촬영 대상 | 권장 방향 | 현재 사용 예정 위치 |
| --- | --- | --- | --- |
| `office-1.jpg` | 본사 외부 또는 승인된 개소식 장면 | 가로형 | 신규 사업장 공지, 필요 시 본사 소개 |
| `office-2.jpg` | 본사 내부·작업공간 | 가로형 | 작업환경 소개, 본사 내부 공간 소개 |
| `minhwa-1.jpg` | 손·붓·도안 중심의 민화 근접 장면 | 가로형 또는 정방형 | 민화 목록 카드 |
| `minhwa-2.jpg` | 여러 사람이 함께하는 민화 작업 장면 | 가로형 또는 정방형 | 민화 상세 대표, 민화 활동 글 |

파일명은 영문 소문자·숫자·하이픈만 사용합니다. 같은 역할의 사진을 추가할 때는 담당 개발자와 번호 규칙을 먼저 확인합니다.

> 주의: 과거 공개 부적합 `images/packing-2.jpg` 파일은 저장소에서 제거되었습니다. 새 `packing-2.jpg`는 위 표의 **새 촬영 예정 파일명**이며, 승인 사진이 수령되기 전에는 홈페이지 이미지 경로로 연결하지 않습니다.

## 3. 사진별 촬영 장면

- 본사 외부·내부: 정돈된 공간과 작업 흐름이 보이도록 촬영하고, 문서·명찰·개인정보가 보이지 않게 합니다.
- 민화: 손과 작업물을 중심으로 촬영합니다. 얼굴 공개 동의가 불분명하면 얼굴을 담지 않습니다.
- 포장·검수: 작업 공정과 안전한 작업 환경이 보이게 하되 송장·주소·주문정보·라벨의 개인 정보가 보이지 않게 합니다.
- 기업 협력·지역 활동: 기업명·로고·인물의 외부 공개 승인을 확인한 실제 사례만 촬영합니다.

## 5. 개인정보·보안 주의사항

- 촬영 전 인물의 외부 공개 동의와 사용 범위를 확인합니다.
- 주민등록번호, 주소, 전화번호, 주문·송장 정보, 사내 문서, 모니터 화면, 계약 자료가 보이지 않게 합니다.
- 기업명·로고·방문객·차량번호는 공개 승인 전에는 식별되지 않게 처리하거나 촬영에서 제외합니다.
- 원본의 촬영일·제공자·승인 근거·내부 보관 위치는 공개 저장소가 아닌 내부 관리 목록에 기록합니다.

## 6. 사진 전달 방법

1. 촬영 담당자는 승인 전 원본을 회사의 승인된 내부 저장공간에 보관합니다.
2. 이 가이드의 통제 파일명으로 복사본을 준비하고, 촬영 대상·공개 동의 상태·제공일을 내부 담당자에게 전달합니다.
3. 개발자는 공개 승인과 품질을 확인한 뒤에만 웹용 파일을 `images/`에 추가합니다.
4. 승인되지 않은 원본, 대용량 편집본, 중복본은 공개 저장소에 올리지 않습니다.

## 7. 사진 수령 후 교체 절차

1. 원본 보관
2. 파일명 확인
3. 크기와 방향 확인
4. 개인정보 노출 확인
5. `images/` 폴더 추가
6. `content.js` 또는 HTML 경로 연결
7. 안내박스 제거
8. alt 문구를 실제 사진에 맞게 작성
9. Preview 확인
10. 승인 후 병합

사진이 없는 동안에는 `thumb`, `hero`, `gallery`에 존재하지 않는 파일 경로를 넣지 않고 안내박스를 유지합니다.
