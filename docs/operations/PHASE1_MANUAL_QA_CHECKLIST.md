# Phase 1 수동 검수 체크리스트

운영 데이터가 아닌 비운영 환경과 가상 계정으로만 확인한다. 문제 기록에는 역할·기기·시간·재현 순서만 남기고 실제 직원 정보는 적지 않는다.

## 공통 환경

- [ ] Android Chrome (360px 폭 포함)
- [ ] Samsung Internet
- [ ] iPhone Safari
- [ ] Windows Chrome
- [ ] Windows Edge
- [ ] 화면 회전 후 가로 스크롤 없음
- [ ] 확대 200%에서 겹침 없음
- [ ] Tab / Shift+Tab / Enter / Space 사용 가능, 포커스 표시 확인
- [ ] `/app/`에만 `비운영 검수환경` 표시, 공개 홈페이지에는 미표시

## 인증·계정

- [ ] 가상 계정 가입 후 `승인 대기` 안내 표시
- [ ] 승인 대기 계정의 `/app/` 직접 접근 차단
- [ ] 최고관리자의 부서·직책·역할 승인 동작
- [ ] 승인 계정의 역할별 홈 이동
- [ ] 로그아웃 후 뒤로가기 차단
- [ ] `suspended`, `departed`, `deleted` 계정의 새로고침 차단
- [ ] 마지막 최고관리자 상태·역할 변경 거부
- [ ] 세션 만료 시 쉬운 로그인 안내

## 일반 근로자

- [ ] 오늘의 업무, 작업방법, 내 일정, 중요공지, 자주 보는 안내 모두 열기
- [ ] 모바일 관리자 메뉴·사이드바 미노출
- [ ] PC에서도 단일 열·고정폭 유지
- [ ] 빈 상태·오류 상태가 쉬운 문장으로 표시
- [ ] 작업방법 단계 이동, 일정 상세, 공지 확인, 안내 필터·검색·상세·뒤로가기 확인
- [ ] 잘못된 ID·대상 제외 자료가 내용 없이 차단

## 관리자·보안·접근성

- [ ] 역할·사용자명·현재 메뉴·사이드바 열기/닫기 확인
- [ ] 운영총괄의 일정·공지·작업방법·안내 관리 확인
- [ ] 팀장의 부서 범위, 현장 책임자의 작업반 범위, 일반 직원의 관리 차단 확인
- [ ] 저장 중 중복 클릭 방지, 실패 후 입력값과 오류 문구 확인
- [ ] KST 00:00 전후의 오늘·내일·적용기간·작업반 멤버십 확인
- [ ] 타 부서·작업반·개인, 초안, 사용 중지, 기간 밖 자료 차단 확인
- [ ] 스크린리더 제목·버튼 목적·필터 상태·로딩/오류 읽기 순서 확인
- [ ] 오류 화면에 SQL·RPC명·토큰·키가 없음
