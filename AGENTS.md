# 태장 홈페이지 작업 기준

이 저장소는 농업회사법인 태장 주식회사의 공식 홈페이지입니다.

## 현재 기술 구조

- 순수 HTML, CSS, JavaScript로 구성된 정적 사이트입니다.
- 공통 스타일은 `assets/css/styles.css`, 공통 동작은 `assets/js/site.js`에 있습니다.
- 활동·일터 목록 콘텐츠는 `assets/js/content.js`와 `assets/js/listing.js`를 사용합니다.
- Netlify는 저장소 루트를 정적 파일로 배포합니다.

## 이번 기준 정리 작업의 변경 범위

이번 프로젝트 기준 문서 정리 작업에서는 기존 HTML, CSS, JavaScript, 이미지 파일 및 `netlify.toml`을 변경하지 않는다.

이번 작업의 변경 대상은 공동 개발 기준 문서와 관련 문서이며, 기존 정적 홈페이지 파일은 현재 기준본으로 유지합니다.

## 브랜치와 검토 원칙

- `main` 브랜치에 직접 수정하거나 직접 커밋하지 않습니다.
- 모든 변경은 목적이 드러나는 기능 브랜치에서 작업합니다.
- 변경 후 검토 가능한 설명과 테스트 결과를 남기고 PR로 `main` 반영을 요청합니다.
- PR 검토와 사용자 승인이 끝나기 전에는 `main` 병합, 운영 배포, Netlify 설정 변경을 하지 않습니다.

## 수정·보안 원칙

- 요청 범위 밖의 파일은 수정하지 않습니다.
- 사용자 승인 없이 운영 URL, 도메인, Netlify 환경 변수, 배포 설정을 변경하지 않습니다.
- 비밀값은 저장소에 넣지 않습니다.
- 민감정보, 주주명부, 주민등록번호, 계약서 원문, 초상권 사용 여부가 확인되지 않은 사진은 공개 저장소에 올리지 않습니다.

## 콘텐츠와 대외 표현

- 게시 전 사실관계, 날짜, 연락처, 인증·선정 표기, 로고 사용 승인 여부를 확인합니다.
- 태장 공식 정보와 대외 표현 기준은 별도 기준 문서로 관리하며, 홈페이지 문구는 그 문서를 우선합니다.
- 법률·고용·인증 관련 문구는 단정 표현을 피하고 필요한 경우 담당자 또는 공식 자료로 재확인합니다.

## 완료 보고

작업 완료 보고에는 변경 파일, 변경 목적, 확인 결과, 남은 확인 사항을 포함합니다.
